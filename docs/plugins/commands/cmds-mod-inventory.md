# {{ $frontmatter.title }}

### clearinventory

Clear your own inventory.

```
/clearinventory
```

### cleartoolbar

Clear your own toolbar.

```
/cleartoolbar
```

### clearequipment

Clear your own equipment.

```
/clearequipment
```

### getitem

Get an item by name and give to self.

```
/getitem [item_nam]
```

```
/getitem burger
```
